CORE COMPONENTS
================================================================================

1. ORCHESTRATOR AGENT
   Location: src/agents/orchestrator.py
   Purpose: Master coordinator implementing Plan→Reason→Act loop
   Key Methods:
     - execute_task(query, max_iterations)
     - _plan(query, iteration, previous)
     - _reason(plan, query)
     - _act(plan, reasoning)
     - route_task(task, agent_name)

2. RESEARCH AGENT
   Location: src/agents/research_agent.py
   Purpose: Information gathering and academic research
   Key Methods:
     - conduct_research(query, depth)
     - plan_research(query)
     - expand_query(query)
     - extract_citations(sources)
     - evaluate_source_quality(source)

3. REASONING AGENT
   Location: src/agents/reasoning_agent.py
   Purpose: Analysis and logical inference
   Key Methods:
     - chain_of_thought(problem, context)
     - generate_hypotheses(scenario, num_hypotheses)
     - comparative_analysis(items, criteria)
     - fact_check(claims, sources)
     - synthesize_insights(information, theme)

4. LONG-TERM MEMORY
   Location: src/memory/long_term_memory.py
   Purpose: Persistent semantic memory with ChromaDB
   Key Features:
     - Vector embeddings for semantic search
     - Type-based memory categorization
     - Importance scoring and usage tracking
     - Session-based organization
     - Memory consolidation
     - Checkpoint save/load

5. WORKING MEMORY
   Location: src/memory/working_memory.py
   Purpose: Context window management with intelligent pruning
   Key Features:
     - Sliding window for recent items
     - Priority-based memory pruning
     - Token counting and management
     - Task-relevant memory injection
     - Compression to long-term storage

6. TOOL REGISTRY
   Location: src/tools/tool_registry.py
   Purpose: Unified function calling framework
   Key Features:
     - OpenAI-compatible JSON schemas
     - Async execution with error handling
     - Built-in tools (calculator, datetime, memory recall)
     - Extensible registration system

7. CALCULATOR
   Location: src/tools/calculator.py
   Purpose: Mathematical operations
   Operations: add, subtract, multiply, divide, power, sqrt, percentage, eval

8. ARXIV SEARCH
   Location: src/tools/arxiv_search.py
   Purpose: Academic paper retrieval and search
   Features:
     - Paper search with relevance ranking
     - Author and category filtering
     - Recent paper discovery
     - Citation extraction
